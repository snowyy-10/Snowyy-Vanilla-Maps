# Snowyy-Vanilla-Maps
ROUNDS extra levels!

What type of maps to expect?


Expect easy to read maps, and easy to move around. Trying to avoid falling to the void too often.


Maps will also be vertically symmetrical for fights to be a little bit more balanced, just like Vanilla.


The maps do support 8 players, but they are too small for it to work properly. I personally recommend playing with 2 - 4 players, I can't test with more players as my friend group is not that big.
